# 知乎排版助手

一款专为知乎创作者打造的纯前端一键排版工具，将原始草稿自动转换为符合知乎编辑器规则的精美富文本。

## 功能特性

- **一键复制排版后的富文本**：粘贴到知乎编辑器即生效
- **标题识别**：支持 `#`、中文序号（一、/ 第二章 / 1.）、短行标题自动识别
- **Top500 关键词加粗**：内置知乎热门领域高频词库，自动加粗
- **短句换行**：句号/问号/感叹号/分号后自动换行，符合知乎阅读节奏
- **Markdown 双模式输出**：富文本预览 + Markdown 源码，一键切换
- **LaTeX 标记**：变色、边框底色、单线划线、框住四种样式
- **贴合 Markdown 规则**：不输出知乎不支持的 HTML 标签
- **纯前端**：数据不出浏览器，安全隐私

## 技术栈

- React 18 + TypeScript + Vite
- Tailwind CSS 3
- Zustand 状态管理
- lucide-react 图标

## 本地开发

```bash
npm install
npm run dev      # 开发
npm run build    # 构建
npm run check    # 类型检查
```

## 部署

自动部署至 GitHub Pages：https://anderson6666.github.io/paiban/

推送到 `main` 分支即触发 GitHub Actions 自动构建并发布。

## 使用说明

1. 在左侧输入框粘贴草稿
2. 点击「一键排版」按钮
3. 选中文字后点击 LaTeX 按钮可应用标记（变色 / 边框 / 下划线 / 框住）
4. 在右侧预览效果
5. 点击「复制富文本」按钮，粘贴到知乎编辑器即可
